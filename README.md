# 📖 Contact Book

A **simple and efficient** contact book application built with modern web technologies.

## 🚀 Tech Stack
This project leverages the following technologies:

- ⚛ **React** – Declarative and efficient UI library  
- 🏗 **TypeScript** – Type-safe development  
- ⚡ **Vite** – Fast and optimized build tool  
- 🌿 **Zustand** – Lightweight state management  
- 🛢 **Supabase** – Backend as a service (BaaS)  
- 🎨 **PrimeReact** – Rich UI components for React  
- 🎨 **Tailwind CSS** – Utility-first CSS framework

## 🛠️ Installation & Setup
Follow these steps to run the project locally:

```sh
# Clone the repository
git clone https://github.com/amitpatil321/contact-book.git

# Navigate to the project directory
cd contact-book

# Install dependencies
npm install

# Start the development server
npm run dev
```
